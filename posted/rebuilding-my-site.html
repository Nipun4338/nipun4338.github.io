Rebuilding My Site with a Mac Terminal Look

I gave the site a fresh coat of paint this week — a macOS terminal aesthetic. The hero is now a real zsh session, every section floats like a Mac window on a dark desktop, and the navigation reads like the system menu bar.

What changed

  • The three duplicated scripts became one shared js/main.js.
  • jQuery and Bootstrap are on current, patched versions (with SRI).
  • Blog posts are now written in Markdown and generated with one command.

The best part: I can write a post like this one in plain text and run npm run new-post -- posts/whatever.md.

Writing the next one

  1. Drop a .md file in posts/.
  2. Run the generator.
  3. npm run build, then npm run deploy.

That's it — back to taking photos.
